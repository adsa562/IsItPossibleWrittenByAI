# IsItPossibleWrittenByAI

這是一個可以在離線狀態下，依照詞彙多樣性、句長分布與波動、可讀性、重複度、粗略困惑度等基本觀點分析文章產自AI可能性的小程式。

你可以將要分析的.txt與.docx文件放置於同一個資料夾內，讓這個小程式利用AI生成文章時的常見邏輯來分析這些文章是否可能由AI生成，並輸出一份Excel格式的統計報告與PNG長條圖表供檢視。

測試用的文本採用了幾個大語言模型，請他們各自生成維基百科的300字介紹，以及維基百科和某知名內容農場的文章各自擷取下來作為範例之用。

此外在執行Release出來的單一exe檔時，會額外跳出一個命令提示字元視窗，請不要關閉且忽略它。

📊注意：本工具僅供參考，請勿作為唯一判斷AI文筆與否的依據。📊

輸出解讀參考：

* 0–40％ → 很可能是人類撰寫
* 40–70％ → 模糊區（可能混合）
* 70–100％ → 高機率為 AI 生成
